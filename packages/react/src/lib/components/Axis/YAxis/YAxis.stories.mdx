# Axis Component

This `<YAxis>` component allows you to define a vertical Axis on a chart. This is a utility component that combines both an <LinkTo kind="components-axis-types--string-axis#axis-component">`<Axis>`</LinkTo> component, along with a `<AutoScale>` component. It is recommended to use this component where possible to save duplicate configuration for a `<AutoScale>`.

Please see the <LinkTo kind="components-axis-types--string-axis#axis-component">Axis</LinkTo> component for more information on it's capabilities and configuration.

### Props

| Prop            | Type            | Default    | Note                                                                                                                                                     |
| --------------- | --------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `aggregate`     | `boolean`       | false      | Whether values in `fields` should be summed when determining the scale                                                                                   |
| \*`fields`\*\*  | `Array<string>` | []         | The set of fields that this axis represents. These should correspond to the fields being displayed on this Axis within the data.                         |
| `position`      | `string`        | `"bottom"` | The position of the axis. One of `top` or `bottom`. This will use the appropriate [d3.axis](https://github.com/d3/d3-axis#axisTop) orientation function. |
| `scaleType`     | `string`        | null       | The type of scale being used. Should be one of `linear`, `time`, `power`, `log`, `band`, `point`                                                         |
| `showGridlines` | `boolean`       | true       | Determines whether gridlines should be shown against this axis.                                                                                          |
| `tickFormat`    | `function`      | null       | See https://github.com/d3/d3-axis#axis_tickFormat                                                                                                        |
| `ticks`         | `function`      | null       | See https://github.com/d3/d3-axis#axis_ticks                                                                                                             |
| `tickSizeInner` | `number`        | 6          | See https://github.com/d3/d3-axis#axis_tickSizeInner.                                                                                                    |
| `tickSizeOuter` | `number`        | 6          | See https://github.com/d3/d3-axis#axis_tickSizeOuter.                                                                                                    |
| `tickPadding`   | `number`        | 3          | See https://github.com/d3/d3-axis#axis_tickPadding.                                                                                                      |
| `title`         | `string`        | null       | The title of the axis. This will default to the value in field(s)                                                                                        |
