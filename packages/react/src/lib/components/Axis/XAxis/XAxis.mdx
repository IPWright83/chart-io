# X Axis Component

This `<XAxis>` component allows you to define a horizontal Axis on a chart. This is a utility component that combines both an [Axis](?path=/docs/components-axis--docs) component, along with an `<AutoScale>` component. It is recommended to use this component where possible to save duplicate configuration for an `<AutoScale>`.

Please see the [Axis](?path=/docs/components-axis--docs) component for more information on it's capabilities and configuration.

### Props

| Prop            | Type                        | Default | Note                                                                                                                                                     |
| --------------- | --------------------------- | ------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `aggregate`     | `boolean`                   | `false`   | Whether the scale should be aggregated (used for stacked charts)                                                                                         |
| `domain`        | `Array<number/string/date>` | `null`    | An optional override of the domain for the scale. This will replace any auto calculated domain. See https://github.com/d3/d3-scale#continuous_domain     |
| **`fields`\***  | `string/Array<string>`      |         | The field or set of fields that this axis represents. These should correspond to the fields being displayed on this Axis within the data.                |
| `position`      | `string`                    | `"bottom"`  | The position of the axis. One of `bottom` or `top`. This will use the appropriate [d3.axis](https://github.com/d3/d3-axis#axisTop) orientation function. |
| `scaleType`     | `string`                    | `null`    | An optional override of the type of scale that should be used. One of "linear", "time", "power", "log", "band", "point"                                  |
| `showGridlines` | `boolean`                   | `true`    | Determines whether gridlines should be shown against this axis.                                                                                          |
| `tickFormat`    | `function`                  | `null`    | See https://github.com/d3/d3-axis#axis_tickFormat                                                                                                        |
| `tickSizeInner` | `number`                    | `6`       | See https://github.com/d3/d3-axis#axis_tickSizeInner.                                                                                                    |
| `tickSizeOuter` | `number`                    | `6`       | See https://github.com/d3/d3-axis#axis_tickSizeOuter.                                                                                                    |
| `tickPadding`   | `number`                    | `3`       | See https://github.com/d3/d3-axis#axis_tickPadding.                                                                                                      |
| `ticks`         | `any[]`                     | `null`    | See https://github.com/d3/d3-axis#axis_ticks                                                                                                             |
| `tickValues`    | `string[]`                  | `null`    | See https://github.com/d3/d3-axis#axis_tickValues                                                                                                        |
| `title`         | `string`                    | `null`    | The title of the axis. This will default to the value in field(s)                                                                                        |
