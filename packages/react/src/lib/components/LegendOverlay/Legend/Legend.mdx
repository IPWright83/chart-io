# Legend Component

The `<Legend>` component allows us to create a legend containing multiple entries, which is useful for showing the information about the series forming a chart.

<Canvas>
    <Story id="components-legendoverlay-legend--default" />
</Canvas>

The legend itself displays rows of <LinkTo kind="components-legendoverlay-legend--scatter">`<LegendItem>`</LinkTo> components which offer their own configuration.

### `<Legend>` Props

> **Note:** Normally you don't interact with the Legend component directly, instead letting the <LinkTo kind="components-legendoverlay--default">`<LegendOverlay>`</LinkTo> component handle the displaying of the tooltip which provides all the required props.

| Prop            | Type     | Default   | Note                                                                                                          |
| --------------- | -------- | --------- | ------------------------------------------------------------------------------------------------------------- |
| `items`         | `array`  | undefined | The list if items that should be renders as `<LegendItem>`s                                                   |
| `positionStyle` | `object` | {}        | Additional css to control the position, typically containing just the top, left, bottom, right css properties |
| `formatters`    | `object` | {}        | A keyed collection of format functions `(name: string) => string`                                             |
