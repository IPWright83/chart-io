# TooltipItem Component

A TooltipItem components represents an entry within the Tooltip. Typically this is a value from a series in a chart. For a known series a shape will be returned, with appropraite styling to reflect the current entry. For example:

<Canvas of={TooltipItemStories.Scatter} />

It is also possible to have a tooltip item that is simply a value, which doesn't have a series to represent it. For example this might represent the value on an x-axis line chart.

<Canvas of={TooltipItemStories.Value} />

There are a number of different shapes avaliable for use:

<Canvas of={TooltipItemStories.Line} />
<Canvas of={TooltipItemStories.Bar} />
<Canvas of={TooltipItemStories.Scatter} />

### Props

| Prop        | Type       | Default     | Note                                                                                      |
| ----------- | ---------- | ----------- | ----------------------------------------------------------------------------------------- |
| **`name*`** | `string`   |             | The name of the value/series being shown in the tooltip                                   |
| `value`     | primitive  |             | The value of the point to display in the tooltip                                          |
| `icon`      | `string`   |             | The type of the shape. One of ["circle", "square", "line", "none"]                        |
| **`fill`**  | `string`   |             | The colour of the series                                                                  |
| `prefix`    | `string`   | `undefined` | A string to prefix the tooltip value with                                                 |
| `suffix`    | `string`   | `undefined` | A string to suffix the tooltip value with                                                 |
| `format`    | `function` | `undefined` | A function that returns a custom tooltip value. `(name: string, value: string) => string` |
