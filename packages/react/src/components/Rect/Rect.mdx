# Rect Component

The `<Rect>` component allows us to define a layer to render a custom rectangle either underneath or on top of the plots. This can be useful for defining threshold bands in the background, along with a variety of other use cases.

<Canvas>
    <Story id="components-rect--default" />
</Canvas>

This can be especially useful when combined with the `chartSelectors` to obtain the scale.

### Props

| Prop           | Type     | Default | Note                                                                                     |
| -------------- | -------- | ------- | ---------------------------------------------------------------------------------------- |
| **`x`\***      | `number` |         | The x position of the rect https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/x  |
| **`y`\***      | `number` |         | The y position of the rect https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/y  |
| **`width`\***  | `number` |         | The width of the rect https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/width   |
| **`height`\*** | `number` |         | The height of the rect https://developer.mozilla.org/en-US/docs/Web/SVG/Attribute/height |
| `fill`         | `string` |         | An optional fill colour to apply to the polygon                                          |
| `stroke`       | `string` |         | An optional stroke color to apply to the polygon                                         |
| `opacity`      | `number` |         | The opacity of the polygon                                                               |
