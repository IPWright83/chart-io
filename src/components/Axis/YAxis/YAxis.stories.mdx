# Axis Component

This `<YAxis>` component allows you to define a vertical Axis on a chart. This is a utility component that combines both an `<Axis>` component, along with a `<Scale>` component. It is recommended to use this component where possible to save duplicate configuration for a `<Scale>`.

Please see the <LinkTo kind="Axis-Axis">Axis</LinkTo> component for more information on it's capabilities and configuration.

### Props

| Prop        | Type            | Default | Note                                                                                                                                                    |
| ----------- | --------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `position`  | `string`        |         | The position of the axis. One of `left`, `right`. This will use the appropriate [d3.axis](https://github.com/d3/d3-axis#axisLeft) orientation function. |
| `fields`    | `Array<string>` |         | The set of fields that this axis represents. These should correspond to the fields being displayed on this Axis within the data.                        |
| `scaleType` | `string`        | null    | The type of scale being used. Should be one of `linear`, `time`, `power`, `log`, `band`, `point`                                                        |
| `aggregate` | `boolean`       | false   | Whether values in `fields` should be summed when determining the scale                                                                                  |
